# Анализ цифровой зрелости и ИТ-стратегия

Проект оценивает цифровую зрелость подразделений компании и преобразует результаты опроса в приоритизированный план ИТ-инвестиций.

## Задача

Выявить подразделения с наибольшим цифровым разрывом, определить слабые направления и предложить типовые инициативы с ожидаемой стоимостью и эффектом.

## Данные

`digital_maturity_survey.csv` содержит 80 наблюдений по пяти полям:

- `department` — подразделение;
- `data_quality` — качество данных;
- `automation_level` — уровень автоматизации;
- `analytics_capability` — аналитические компетенции;
- `it_infra_rating` — состояние ИТ-инфраструктуры.

Все четыре оценки заданы по шкале от 1 до 5.

## Методика

1. Проверка пропусков, аномалий и допустимого диапазона оценок.
2. Расчёт среднего `maturity_score` по четырём направлениям.
3. Агрегация результатов по подразделениям.
4. Классификация зрелости: низкая (до 2,5), средняя (2,6–3,8) и высокая (выше 3,8).
5. Анализ цифровых разрывов и формирование рекомендаций.
6. Корреляционный анализ показателей зрелости.
7. Расчёт инвестиционного приоритета с поправкой на размер подразделения.

## Технологии

Python, pandas, NumPy, Matplotlib, seaborn, Jupyter Notebook.

## Структура

- `strategy.ipynb` — расчёты, графики и рекомендации;
- `strategy.html` — статическая версия анализа;
- `digital_maturity_survey.csv` — результаты опроса;
- `ПРАКТИЧЕCКАЯ РАБОТА 3.pdf` — оформленный отчёт.
