# Формирование портфеля ИТ-проектов

Аналитический проект по ранжированию ИТ-инициатив и формированию инвестиционного портфеля с учётом доходности, риска, стратегического соответствия, длительности и бюджетного ограничения.

## Задача

Перевести неоднородные параметры проектов в прозрачную систему приоритетов и выбрать набор инициатив, который сочетает экономический эффект со стратегической ценностью.

## Данные

`it_portfolio_projects.csv` содержит 200 проектов и шесть полей:

- `project_name` — название;
- `investment` — необходимые инвестиции;
- `expected_revenue` — ожидаемая выручка или экономия за три года;
- `risk_score` — риск по шкале 1–10;
- `alignment_score` — соответствие стратегии по шкале 1–5;
- `duration_months` — длительность в месяцах.

## Методика

1. Проверка типов, пропусков и диапазонов значений.
2. Расчёт ROI для оценки эффективности вложений.
3. Расчёт интегрального `Value Index`, объединяющего доходность, риск, стратегическое соответствие и длительность.
4. Построение пузырьковой диаграммы «риск — доходность», где размер отражает инвестиции, а цвет — стратегическое соответствие.
5. Ранжирование проектов и формирование топ-портфеля.
6. Проверка портфеля при бюджетном ограничении 15 млн рублей.
7. Использование линейной оптимизации для поддержки выбора.

## Технологии

Python, pandas, NumPy, Matplotlib, SciPy, Jupyter Notebook.
