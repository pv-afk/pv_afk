# Кредитный скоринг

ML-проект для многоклассовой оценки результата кредитной заявки на основании кредитной истории, поведения клиента и анкетных характеристик. Проект объединяет две витрины данных, выполняет разведочный анализ и сравнивает линейную модель с градиентным бустингом.

## Задача

Построить модель классификации целевого признака `Approved_Flag`, уделив внимание пропускам, категориальным данным, дисбалансу классов и качеству на отложенной выборке.

## Данные

Обе таблицы содержат по 51 336 строк и объединяются по `PROSPECTID`.

- `credit_risk_file_1.csv` — структура кредитного портфеля: активные и закрытые договоры, типы кредитов, давность открытия и платежная дисциплина;
- `credit_risk_file_2.csv` — просрочки, запросы кредитной истории, демография, доход, занятость, утилизация лимитов, кредитный балл и целевой класс.

## Этапы работы

1. Объединение источников и проверка качества данных.
2. Анализ распределения целевых классов и признаков.
3. Кодирование целевой переменной и удаление дублирующих/сильно коррелирующих признаков.
4. Разделение на train/test со стратификацией.
5. Препроцессинг числовых и категориальных признаков: заполнение пропусков, масштабирование и one-hot encoding.
6. Визуальная диагностика пространства признаков с PCA.
7. Балансировка обучающей выборки средствами `imbalanced-learn`.
8. Подбор гиперпараметров Logistic Regression и XGBoost через `RandomizedSearchCV`.
9. Оценка качества по F1-score и `classification_report`.

## Технологии

Python, pandas, NumPy, seaborn, scikit-learn, imbalanced-learn, XGBoost, Jupyter Notebook.

## Структура

- `credit_scoring.ipynb` — полный ML-анализ;
- `credit_scoring.html` — статическая версия ноутбука с сохранёнными результатами;
- `credit_risk_file_1.csv`, `credit_risk_file_2.csv` — исходные таблицы.
