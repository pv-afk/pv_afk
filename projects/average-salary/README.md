# Прогноз средней заработной платы

Проект посвящён анализу временного ряда средней заработной платы и построению прогноза на 36 месяцев. В работе показан полный статистический цикл: от проверки свойств ряда до выбора и применения сезонной модели.

## Задача

Исследовать динамику заработной платы, выявить тренд и сезонность, добиться стационарности ряда и получить интерпретируемый среднесрочный прогноз.

## Данные

Файл `salary.csv` содержит 399 ежемесячных наблюдений:

- `month` — месяц наблюдения;
- `salary` — средняя заработная плата.

Разделитель CSV — точка с запятой. Дата используется как индекс временного ряда.

## Методика

1. Загрузка, первичная проверка и визуальный анализ данных.
2. Исследование распределения и сезонной структуры ряда.
3. Преобразование Бокса—Кокса для стабилизации дисперсии.
4. Дифференцирование для устранения тренда и сезонности.
5. Подбор параметров ARIMA с помощью `auto_arima`.
6. Сравнение кандидатов и обучение SARIMAX.
7. Обратное преобразование прогноза в исходную шкалу.
8. Формирование прогноза на следующие 36 месяцев.

## Технологии

Python, pandas, Matplotlib, seaborn, SciPy, statsmodels, pmdarima, Jupyter Notebook.
