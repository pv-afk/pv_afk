# Происхождение материалов

Подбор тем вдохновлён открытым репозиторием [DA_and_ML_Python](https://github.com/Filareth2015/DA_and_ML_Python/tree/master), состояние которого было просмотрено на коммите `bca40208f1361a109471cd2dcb7e09a75bb0ffe4`.

В исходном репозитории не указана лицензия. Поэтому его код, данные, формулировки заданий и содержимое ноутбуков в этот проект не копировались. Реализация, структура, тексты и примеры написаны заново с использованием публичных API `scikit-learn` и самостоятельно составленного текстового корпуса.

Из исходного списка тем сохранены только общие учебные направления: линейные модели, бинарная классификация, ансамбли, снижение размерности и обработка текста.
