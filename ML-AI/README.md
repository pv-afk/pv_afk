# ML/AI

Базовый учебный набор для подготовки к junior-позициям ML/AI/LLM Engineer. Файловая структура повторяет выбранные разделы исходного репозитория: пронумерованные ноутбуки и используемые ими датасеты находятся на одном уровне.

## Ноутбуки

| Ноутбук | Тема |
|---|---|
| `1.1. Линейная регрессия_.ipynb` | Линейная модель, Ridge, метрики и анализ остатков |
| `1.2. Линейная регрессия - переобучение и регуляризация.ipynb` | Полиномиальные признаки, переобучение, Ridge и Lasso |
| `1.3. Линейная регрессия и SGD.ipynb` | Сравнение аналитической регрессии и SGD |
| `1.4. Логистическая регрессия для задачи бинарной классификации.ipynb` | Pipeline, Logistic Regression, ROC-AUC и confusion matrix |
| `1.5. Bagging & Randon Forest.ipynb` | Деревья, Random Forest, boosting и кросс-валидация |
| `1.6. Градиентный бустинг.ipynb` | Learning rate, число деревьев, ROC-AUC и feature importance |
| `1.7. Бинарная классификация.ipynb` | Пороги, precision, recall, F1, ROC и PR-кривые |
| `1.8. 1NN vs RandomForest.ipynb` | Сравнение 1NN и Random Forest на распознавании цифр |
| `4.1. Размещение баннеров (кластеризация).ipynb` | K-Means, метод локтя и центры кластеров |
| `4.2. Визуализация данных.ipynb` | EDA, распределения, boxplot, pairplot и корреляции |
| `4.3. PCA.ipynb` | Explained variance, проекция и нагрузки признаков |
| `4.4. LDA.ipynb` | Тематическое моделирование и интерпретация тем |
| `4.5. Поиск контекста для LLM.ipynb` | TF-IDF retrieval, cosine similarity и сбор контекста для RAG |
